# Ölüler İçin Enternasyonal

*Cahit Koytak 11.02.2013*

<div class="yazi"><p>bize katılsanıza, yersizler yurtsuzlar,<br/>bize katılsanıza, dilsizler, yolsuzlar, çulsuzlar!<br/>bize katılsanıza! <br/>biz, yeryüzünün fukara ölüleri,<br/>cenneti aramaya gidiyoruz.<br/>çullarınızı silkip, küllerinizi saçıp<br/>bize katılsanıza!</p>
<p>bize katılsanıza, siz toynakların altındakiler!<br/>tekerlerin altındakiler, <br/>ayakların altındakiler,<br/>küfelerin, balyaların,                 <br/>                 kasaların altındakiler!</p>
<p>bize katılsanıza, deliler, divaneler,<br/>âşıklar, aylaklar, işsizler,<br/>bize katılsanıza,<br/>yaşlı kurtlar, yitik kuzular <br/>ve sürüsünü sele kaptırmış çoban!</p>
<p>bize katılsanıza, biz, yeryüzünün                  <br/>                 fukara ölüleri, biçare ölüleri, <br/>bize katılsanıza, biz pusuda ölen asker,<br/>dağda vurulan gerilla,<br/>sınırda bombalanan kaçakçı,<br/>biz, kardeşleriniz, kuzenleriniz,                 <br/>                 yeğenleriniz!</p>
<p>ne çabuk unuttunuz kardeşiniz Gandi’yi,<br/>ne çabuk unuttunuz Hasan’la Hüseyn’i,<br/>ne çabuk, ne çabuk unuttunuz<br/>kardeşiniz İsa’yı, kardeşiniz Musa’yı,               <br/>                 kardeşiniz Muhammed’i!<br/>ne çabuk unuttunuz bütün o ağlayan anaları<br/>Asiye’yi, Meryem’i, Fatıma’yı,<br/>bizim için ağlayarak bekletenler, kıyameti,<br/>o sonsuz anaları,<br/>o uçsuz bucaksız anaları!</p>
<p>bize katılsanıza, bize,<br/>biz yeryüzünün fukara ölüleri,<br/>arayıp bulmaya gidiyoruz, bakın,<br/>çitlerle, dikenli tellerle, tabyalarla<br/>ırklara, dinlere, mezheplere ayrılmış, <br/>insana kapatılmış bütün o cennetleri, <br/>beylerin, paşaların,           <br/>                 padişahların has bahçelerini;</p>
<p>arayıp bulmaya ve yakmaya değil,<br/>arayıp bulmaya ve yıkmaya değil,<br/>arayıp bulmaya ve muhabbet            <br/>                 ekmeye gidiyoruz <br/>mayın tarlalarına, o cennetlerdeki;<br/>muhabbet ekmeye           <br/>                 ve dostluk biçmeye... </p>
<p>yumruklarımızı ve naralarımızı            <br/>                 toprağa gömüp<br/>güller, çiçekler açmaya gidiyoruz;<br/>rüzgâr olup estirmeye gidiyoruz            <br/>                 çığlıklarımızla;<br/>şimşek olup çakmaya            <br/>                 fikirlerimizle, buluşlarımızla;</p>
<p>yağmurlar yağdırmaya gönüllerimizden,<br/>çavlanlar akıtmaya dillerimizden <br/>ve dereler ve ırmaklar,            <br/>                 türkülerimizden!</p>
